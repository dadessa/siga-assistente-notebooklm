# SIGA — Assistente Especializado com NotebookLM

## 📌 Sobre o Projeto

Este projeto apresenta o SIGA (Sistema de Inteligência e Gestão), uma plataforma de monitoramento de redes, portais, rádio e TV, com apoio de inteligência artificial para tomada de decisão estratégica.

O objetivo foi criar um assistente especializado utilizando o Google NotebookLM, capaz de responder perguntas com base exclusivamente no documento do produto.

---

## 🧠 Produto Criado

SIGA — Sistema de Inteligência e Gestão

O SIGA é uma plataforma que:
- monitora redes sociais, portais, rádio e TV
- identifica narrativas e crises
- gera alertas em tempo real
- apoia gestores públicos na tomada de decisão

## 🎯 Objetivo do Assistente

O assistente atua como especialista no SIGA, sendo capaz de:

- explicar o produto
- descrever funcionalidades
- detalhar o fluxo de decisão
- apresentar casos de uso
- responder perguntas com base no documento
